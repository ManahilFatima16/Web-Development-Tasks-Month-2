# Web-Development-Tasks-Month-2
# Web Music Player

## Overview
This is a simple web-based music player built using HTML, CSS, and JavaScript.  
It allows users to play, pause, and switch between songs.

## How to Run
1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).  
2. Click play to listen to the songs.  
3. Navigate through the songs if multiple are available.



# Social Network Platform

## Overview
This is a simple web-based social network platform using Node.js, HTML, CSS, and JavaScript.  
Users can register, login, and navigate through home page content.

## How to Run
1. Run `npm install` to install dependencies.
2. Start the server: `node server.js`
3. Open browser and go to `http://localhost:3000`
